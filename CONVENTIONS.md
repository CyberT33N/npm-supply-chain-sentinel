# Conventions

`CONVENTIONS.md` is the workspace-level table of contents for normative repository conventions.

The canonical implementation and review rules live in the leaf slices under `conventions/`.
Descriptive documentation in `docs/` must reference these convention slices instead of re-declaring policy.
